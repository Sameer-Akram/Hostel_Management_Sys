# Hostel Management system helps hostel warden to manages records of hostel in an efficient manner.
# Features of App
# 1-Student View
# 2-Warden view
# 3-Authentication
# 4-Forget Password Mail
# 5-Email Verification Mail
# 6-Attendance
# 7-Students Request
# 8-Mess charges
# 9-Room Allocation
# 10-Notice with (Notification)
